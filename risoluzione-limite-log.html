Risoluzione: Limite notevole del logaritmo

Esempio:

lim x → 0 (ln(1 + x) / x)

Risoluzione:

Per calcolare il limite, consideriamo l'espressione:

ln(1 + x) = ln[(1 + x)(1 - x) / (1 - x)]

Riscrivendo l'argomento del logaritmo sfruttando il denominatore coniugato:

ln(1 + x) = ln((1 - x + x - x²) / (1 - x)) = ln((1 - x²) / (1 - x))

Utilizzando la proprietà del logaritmo:

ln(1 + x) = ln(1 - x²) - ln(1 - x)

Per valori di x prossimi a zero, possiamo approssimare ln(1 - x) ≈ -x e ln(1 - x²) ≈ -x².

ln(1 + x) ≈ -x² + x

Ora possiamo dividere per x:

(ln(1 + x) / x) ≈ (-x² + x) / x = 1 - x

Facendo tendere x a 0, otteniamo il risultato finale:

lim x → 0 (ln(1 + x) / x) = 1
