Risoluzione: Limiti di radice quadrata

Esempio: lim x → 4 √x = 2

Risoluzione:

Passo 1: Calcolare |√x - 2|
Utilizzando la formula della differenza di quadrati:

|√x - 2| = |x - 4| / |√x + 2|

Passo 2: Imporre la condizione che |√x - 2| < ε

|x - 4| / |√x + 2| < ε

Per x vicino a 4, √x + 2 sarà vicino a 4. Possiamo approssimare |√x + 2| con un valore maggiore o uguale a 4:

|x - 4| / 4 < ε

Passo 3: Troviamo δ
Da |x - 4| / 4 < ε, otteniamo:

|x - 4| < 4ε

Passo 4: Scegliere il valore di δ
δ = 4ε

In altre parole, per ogni ε > 0, possiamo scegliere δ = 4ε per fare in modo che, se 0 < |x - 4| < δ, allora |√x - 2| < ε.

Quindi, il risultato è:

limx → 4 √x = 2
